# Sammlerhausen – Version 3

Sammlerhausen ist der neue Anzeigename des bisherigen ComicArchiv-Projekts. Der Repository-Ordner und die lokale Datenbank behalten absichtlich ihre bisherigen technischen Namen, damit vorhandene Daten erhalten bleiben.

## Update

Den gesamten Inhalt dieses Ordners in das bestehende GitHub-Repository `ComicArchiv` hochladen und vorhandene Dateien ersetzen. Die GitHub-Pages-Einstellungen bleiben unverändert.

Nach dem Deployment die App online öffnen und den eingeblendeten Button **Update** antippen. Vor größeren Änderungen empfiehlt sich ein JSON-Backup.
